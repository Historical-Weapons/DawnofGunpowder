[🧭 What the game is about ](https://historical-weapons.github.io/DawnofGunpowder/) 
The game is a 13th-century tactical strategy war game focused on:

Historical warfare (Mongols, Japan, East Asia)
Army composition and tactics
Terrain-based battles (steppes, oceans, cities)
Unit variety (gunpowder units, crossbows, spearmen, rockets, etc.)
Progression through battles (veterancy system)

The manual text in your code literally describes it as:

a tactical strategy game set in a 13th-century world of conquest and shifting alliances
